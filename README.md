# 🔥 Crowd Density Analyzer (Advanced)

system that analyzes crowd density using computer vision and YOLOv8.

## 🚀 Features
- Real-time people detection
- Zone-based crowd analysis (grid system)
- Heatmap visualization
- Overcrowding alert system
- FPS monitoring

## 🧠 Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Streamlit

## ⚡ Use Cases
- Event crowd monitoring
- Public safety systems
- Smart surveillance

## 📌 How it works
1. Upload video
2. Model detects people
3. Frame divided into zones
4. Density calculated per zone
5. Heatmap + alerts generated
